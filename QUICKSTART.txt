🎯 MANUAL SETUP:

1. Install dependencies:
   npm install

2. Create .env.local file:
   copy .env.local.example .env.local

3. Edit .env.local and add your NewsAPI key:
   NEXT_PUBLIC_NEWS_API_KEY=your_key_here
   NEXT_PUBLIC_USE_OPENSTREETMAP=true

4. Run development server:
   npm run dev

5. Open browser:
   http://localhost:3000

🔧 TROUBLESHOOTING:

Problem: "npm is not recognized"
Solution: Install Node.js from https://nodejs.org/

Problem: "Failed to fetch news"
Solution: Check your API key in .env.local

Problem: Notifications not working
Solution: Click "Enable Alerts" and allow in browser

Problem: Port 3000 in use
Solution: npm run dev -- -p 3001

See SETUP.md for more troubleshooting help.
