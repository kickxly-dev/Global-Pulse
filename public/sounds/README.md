# Notification Sounds

This directory contains notification sound files for Global Pulse.

## Required Sound Files

Place the following MP3 files in this directory:

### Breaking News Sounds
- `breaking.mp3` - Main breaking news alert (default)
- `urgent.mp3` - Urgent news alert
- `siren.mp3` - Siren-style alert

### Local Alert Sounds
- `alert.mp3` - Standard alert chime (default)
- `bell.mp3` - Bell notification
- `ding.mp3` - Simple ding sound

### Personal Topic Sounds
- `notification.mp3` - Soft notification (default)
- `gentle.mp3` - Gentle alert
- `subtle.mp3` - Subtle notification

## Sound Requirements

- Format: MP3
- Duration: 1-3 seconds recommended
- Volume: Normalized to prevent clipping
- Sample Rate: 44.1kHz or 48kHz

## Where to Get Sounds

You can download free notification sounds from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/
- https://soundbible.com/

Or create your own using audio editing software.

## Testing

After adding sound files, test them in the Settings panel of the website.
